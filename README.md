# Automiq Site

This project features an AI voice-bot widget and service demos built with **TanStack Start** and **Vite**.

## Local Development

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Set up environment variables**:
   Copy `.env.example` to `.env` and fill in your API keys:
   ```bash
   cp .env.example .env
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

## Deployment

To deploy this project to Cloudflare Pages, Vercel, Railway, or a self-hosted Node.js server, please follow the step-by-step instructions in the [Deployment Guide (DEPLOY.md)](DEPLOY.md).
